# Sales Analysis Dashboard (2019)

This project presents a sales dashboard for the first quarter of 2019.  
The dashboard provides insights into sales performance using different visualizations.

## Dashboard Features
- Total Sales Overview
- Gross Income Analysis
- Sales by Product Line
- Sales by City
- Monthly Sales Trend
- Product Quantity Distribution
- Sales by Gender

## Tools Used
- Data Analysis
- Data Visualization
- Dashboard Design
